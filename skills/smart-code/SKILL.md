# smart-code

- Mandatory TDD:
  1. Write failing test (RED).
  2. Run and see failure.
  3. Write minimal code to pass (GREEN).
  4. Run and see pass.
  5. Refactor.
  6. Repeat.
- Bugfixes: Write reproducing test first. No production edits without tests.
