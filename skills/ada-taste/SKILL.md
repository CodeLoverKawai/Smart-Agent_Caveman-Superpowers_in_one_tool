# ada-taste

- Overwrite default LLM design biases (generic colors, browser-default fonts, boxy tables).
- UI Requirements:
  - Curated, harmonious HSL/HEX palettes (sleek dark modes, smooth gradients, subtle borders).
  - Modern typography (Inter, Outfit, Roboto) via Google Fonts instead of system defaults.
  - Rich aesthetics: Glassmorphism (`backdrop-filter`), micro-animations, hover transitions.
  - Advanced CSS: Custom properties, Grid, container queries, `:has()`, `:user-valid`.
  - Accessible inputs: Semantic focus rings, proper validation states, autocomplete.
- Zero placeholder elements. Embed actual generated image assets where needed.
- Validate WCAG 2.2 AA contrast and keyboard accessibility (tabindex, focus visible).
