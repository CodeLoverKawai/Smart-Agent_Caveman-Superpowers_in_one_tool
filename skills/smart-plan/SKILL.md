# smart-plan

- Write Nano-Plan to `implementation_plan.md` in conversation folder.
- No prose/storytelling.
- Files impact checklist:
  - `[+] path` (new)
  - `[*] path` (modify)
  - `[-] path` (delete)
- Task checklist:
  - `[ ]` pending
  - `[/]` running
  - `[x]` done
